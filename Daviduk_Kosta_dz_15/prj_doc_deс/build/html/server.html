Server module¶

Серверный модуль мессенджера.

Использование

Модуль подерживает аргементы командной стороки:

  1. -p - Порт на котором принимаются соединения

  2. -a - Адрес с которого принимаются соединения.

Примеры использования:

python server.py

Запуск приложения с параметрами по умолчанию.

python3 server.py -p {порт} -a {имя сервера}

  1. {порт} - порт по которому принимаются подключения

  2. {имя сервера} - адрес сервера сообщений.

Запуск приложения с указанием подключаться к серверу по адресу port:ip_address

server.py¶

Запускаемый модуль,содержит парсер аргументов командной строки и функционал инициализации приложения. Создаёт серверное приложение на основе PyQt5.

server. createParser ()

Парсер аргументов командной строки, определяет 2 элемента:

  • адрес с которого принимать соединения

  • порт
